### Comparator 직접 구현해서 사용하기

```
TreeSet<Integer> setAsc = new TreeSet<>(new Comparator<Integer>() {
    @Override
    public int compare(Integer o1, Integer o2) {
        return o1.compareTo(o2);  // 오름차순
    }
});

TreeSet<Integer> setDesc = new TreeSet<>(new Comparator<Integer>() {
    @Override
    public int compare(Integer o1, Integer o2) {
        return o2.compareTo(o1);  // 내림차순
    }
});

```

### 람다식으로 간단하게 (Java 8+)
```
TreeSet<Integer> setAsc = new TreeSet<>((a, b) -> a - b);  // 오름차순
TreeSet<Integer> setDesc = new TreeSet<>((a, b) -> b - a); // 내림차순
```
